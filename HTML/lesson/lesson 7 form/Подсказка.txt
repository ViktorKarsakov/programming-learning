name                   ->  key pod kotorim znacenie uydet na server ,
                               esli netu name pole ne otpravitsa
    type                   ->  cto za pole
    value                  ->  tekusee znacenie

    required               ->  obyazatelnoe pole

    readonly               ->  tolko dlya ctenie

    maxlength,minlength    ->  oqranicenie po dlinne
    pattern                ->  requlyarnoe virajenie
    list                   ->  svyazka s datalist dlya podskazok
